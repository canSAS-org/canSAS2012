==================================================
Description of the canSAS2012 data format
==================================================

The canSAS (Collective Action for Nomadic Small-Angle Scatterers) met in Uppsala,
Sweden in July 2012 to discuss standardization of the storage of reduced small-angle scattering
of any dimension to be considered for analysis.

This document describes the current specification as an outcome of that meeting.
